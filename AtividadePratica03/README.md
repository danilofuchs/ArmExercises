# Atividade Prática 3 - Interrupções

## Contexto:

Interrupções são utilizadas para permitir aos microcontroladores uma melhor alocação de recursos, liberando-o para a execução de tarefas enquanto os dados úteis dos periféricos ainda não estão disponíveis.

## Tarefas:

Programar uma interrupção por borda de descida na USR_SW1 (PJ0) e uma interrupção por borda de subida na USR_SW2 (PJ1). Para que quando usuário pressione a chave USR_SW1 acenda o LED1 (PN1) e quando soltar a chave USR_SW2 apague o LED1, conforme o exercício do slide 41 do tópico 08.

## Tarefas:

Seguir o passo-a-passo descrito nos slides do tópico 08 a partir do slide 42.
Entregar a pasta do projeto do Keil com todos os arquivos zipado e o vídeo da demonstração na placa. Nomear o arquivo com o nome e o último sobrenome dos dois alunos da dupla. Ex.: fulanodetal1_fulanodetal2_ap3.zip
